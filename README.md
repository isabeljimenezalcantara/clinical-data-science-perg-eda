# Clinical Data Science PERG EDA

Exploratory data analysis of Pattern Electroretinogram (PERG) data to identify clinical patterns across ocular conditions. Focused on data cleaning, visualization, and interpretation of electrophysiological signals in a clinical research context.

Colab Notebook: You can explore the analysis in Google Colab:  
[EDA_PERG.ipynb](https://colab.research.google.com/github/isabeljimenezalcantara/clinical-data-science-perg-eda/blob/main/EDA_PERG.ipynb)

Project Structure
- `data/` : Raw and processed PERG dataset files  
- `notebooks/` : Jupyter/Colab notebooks for EDA  
- `scripts/` : Python scripts for data cleaning, processing, or plotting  
- `figures/` : Generated visualizations from the analysis  

Objectives
1. Load and clean the PERG dataset  
2. Perform exploratory data analysis to identify trends or patterns  
3. Visualize electrophysiological signals in relation to clinical features  
4. Prepare insights for clinical interpretation  

Tools & Libraries
- Python 3.x  
- Pandas, NumPy, Matplotlib, Seaborn  
- Google Colab  

Notes: This repository is part of a clinical data science portfolio showcasing EDA on clinical electrophysiology data.
