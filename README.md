# Clinical Data Science: PERG EDA

## Descripción
Exploratory data analysis of Pattern Electroretinogram (PERG) data to identify clinical patterns across ocular conditions.

## Contenido del repositorio
- `notebooks/` → Jupyter notebooks con análisis paso a paso
- `data/` → Dataset original o muestras (si es público)
- `scripts/` → Funciones o scripts de apoyo
- `figures/` → Visualizaciones generadas en el análisis

## Cómo ejecutar
1. Abrir `notebooks/EDA_PERG.ipynb` en Google Colab
2. Ejecutar las celdas en orden
3. Los gráficos y resultados se guardan en `figures/` (opcional)
